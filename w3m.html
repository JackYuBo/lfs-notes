Using w3m

There are several different console web browsers available for Linux. I have decided that we will use w3m. The w3m web browser works differently than GUI based browsers. Below is some information to help you get started using the browser.

Starting The Browser

You can use w3m to surf the web or use it to open a local HTML file. One of the primary uses for the browser is going to be downloading tar-balls. The location the files are downloaded to is by default the current directory when you started w3m. Therefore, I recommend that you start w3m in the /sources directory. 

w3m http://some.url
w3m path/to/an/html/file

Keyboard Commands

Console based web browsers are keyboard driven, instead of mouse driven. You will need to use several keyboard commands in order to navigate the using w3m.

Hold ShiftShows the gpm pointer and allows you to highlight text
↓ or jMove down a line, but half page down at bottom of screen
↑ or kMove up a line, but half page up at top of screen
EnterFollow a hyperlink
TabSelect next hyperlink
aSave link as. Note, you need to use this to download *.patch files.
Shift-JScroll down one line
Shift-KScroll up one line
Shift-BBack
Ctrl-TOpen link in new tab
Ctrl-QClose current tab
{Move to previous tab
}Move to next tab
Esc-TPop-up tab selection
